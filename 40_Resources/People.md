## Jim Fan
- **名称**: Jim Fan（范麟熙）
- **链接**: https://twitter.com/DrJimFan | https://jimfan.me/
- **简介**: NVIDIA 高级研究科学家，前 OpenAI 研究员，Voyager 论文核心作者之一。专注于 LLM Agent、具身智能与终身学习。
- **为什么重要**: Jim Fan 是 [[Voyager]] 论文的核心推动者，将 LLM 从静态模型转变为能在 Minecraft 中持续学习新技能的终身学习者。他的工作直接定义了"技能库 + 自动课程"的范式，是理解自进化 Agent 工程实现不可绕过的人物。此外，他在社交媒体上的技术评论常常提前预示领域风向。
- **标签**: #people/researcher #type/resource

## Noah Shinn
- **名称**: Noah Shinn
- **链接**: https://noahshinn.com/ | https://github.com/noahshinn024
- **简介**: 麻省理工学院（MIT）博士生，[[Reflexion]] 论文一作。研究聚焦于让 LLM 通过自我反思实现自动纠错和能力提升。
- **为什么重要**: Reflexion 是首个将"自我反思"机制系统化引入 LLM Agent 的论文，提出了 verbal reinforcement learning 的概念。Noah Shinn 的工作证明了 LLM 不需要额外训练就能通过语言层面的自我批评实现性能提升，这是 [[Self-Reflection]] 和 [[Self-Correction]] 概念的直接工程来源。
- **标签**: #people/researcher #type/resource

## Sakana AI
- **名称**: Sakana AI（さかなAI）
- **链接**: https://sakana.ai/
- **简介**: 由前 Google Brain 研究人员 Llion Jones 和 David Ha 创立的东京 AI 实验室。专注于"自然启发智能"——让 AI 像生物进化一样自我组织、自我改进。
- **为什么重要**: Sakana AI 是少数将进化计算与 LLM 深度融合的工业界团队。他们提出的 [[AI-Scientist]] 是第一个能全自动进行科学发现（从读论文到写论文）的系统，代表了自进化 Agent 从玩具任务走向真实科研的里程碑。其"自然启发"哲学也与 [[30_Philosophy/哲学维度 MOC|哲学维度]] 中的生物进化类比高度相关。
- **标签**: #people/team #type/resource

## MetaGPT 团队（DeepWisdom）
- **名称**: MetaGPT 团队 / DeepWisdom
- **链接**: https://github.com/geekan/MetaGPT | https://www.deepwisdom.ai/
- **简介**: 由 Geekan（Alex）领导的开源团队，MetaGPT 是多智能体协作框架的先驱，将软件公司的 SOP 映射到多 Agent 协作中。
- **为什么重要**: MetaGPT 证明了多 Agent 的"角色分工 + 标准化流程"可以产生涌现能力——多个简单 Agent 协作产生的代码质量远超单个 Agent。这是 [[Multi-Agent Co-evolution]] 的工程基石，也是理解"社会进化"（agents 通过协作互相提升）的关键案例。团队在 GitHub 上的活跃维护使其成为实践者的首选参考。
- **标签**: #people/team #type/resource

## Lilian Weng
- **名称**: Lilian Weng（翁丽莲）
- **链接**: https://lilianweng.github.io/
- **简介**: OpenAI 安全研究 VP，前 Applied AI 研究主管。博客以深度、系统、前瞻著称，涵盖 LLM、Agent、AI 安全等主题。
- **为什么重要**: Lilian Weng 的博客文章是领域公认的最佳入门与进阶材料。她的《LLM Powered Autonomous Agents》一文系统梳理了 Agent 架构的演进，被引用数千次。对于自进化 Agents 领域，她的文章提供了从 [[ReAct]] 到 [[Tool Use]] 再到多 Agent 的完整认知地图，且写作质量极高——每篇都是准论文级别的综述。
- **标签**: #people/researcher #type/resource

## Karthik Narasimhan 团队（Princeton NLP）
- **名称**: Karthik Narasimhan / Princeton NLP Group
- **链接**: https://www.karthiknarasimhan.org/ | https://github.com/princeton-nlp
- **简介**: 普林斯顿大学助理教授，ReAct 论文通讯作者，领导 Princeton NLP 实验室。
- **为什么重要**: Karthik Narasimhan 团队是 LLM Agent 架构的奠基者之一。从 [[ReAct]]（推理-行动交替）到 [[Reflexion]]（反思改进），再到后续的多步推理工作，该团队持续产出定义领域的论文。理解自进化 Agents 必须回到 ReAct 范式——它是几乎所有后续工作的基础循环结构。
- **标签**: #people/team #type/resource

## Yejin Choi 团队（Allen Institute / UW）
- **名称**: Yejin Choi / Allen Institute for AI & University of Washington
- **链接**: https://homes.cs.washington.edu/~yejin/
- **简介**: 华盛顿大学教授、Allen Institute for AI 高级研究主任。研究涵盖常识推理、语言生成、AI 安全与价值对齐。
- **为什么重要**: Yejin Choi 团队在常识推理和价值对齐方面的工作，是自进化 Agents 必须面对的"天花板问题"——Agent 可以自我改进能力，但如果改进方向偏离人类价值观，后果严重。她的 [[Symbolic Knowledge Distillation]]、[[Self-Critique]] 相关研究，以及关于 LLM 幻觉和真实性的分析，为 [[Risks]] 和 [[Value Alignment]] 提供了理论基础。

## Google DeepMind（AlphaEvolve / FunSearch 团队）
- **名称**: Google DeepMind — AlphaEvolve / FunSearch 团队
- **链接**: https://deepmind.google/ | https://github.com/google-deepmind/funsearch
- **简介**: DeepMind 的数学与算法发现团队，开发了 FunSearch 和 AlphaEvolve 等基于 LLM 的进化发现系统。
- **为什么重要**: DeepMind 团队代表了"LLM + 进化算法"的最强工业实现。FunSearch 首次用 LLM 发现了新的数学构造（上限集问题），AlphaEvolve 则在矩阵乘法算法上取得突破。这些工作证明了自进化系统不仅能优化已知任务，还能发现人类未知的新知识——这是 [[Recursive Self-Improvement]] 从理论走向实证的关键一步。
- **标签**: #people/team #type/resource
