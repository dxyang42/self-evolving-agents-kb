# 实验区 README

> 本目录存放所有实验记录，遵循统一模板格式。
>
> #type/meta #status/seedling

---

## 📋 实验记录格式

每个实验应为一个独立的 Markdown 文件，命名规则：

```
YYYY-MM-DD_实验主题关键词.md
```

例如：`2025-04-29_记忆机制对比实验.md`

---

## 🗂️ 目录结构

```
20_Engineering/Experiments/
├── README.md          # 本文件
├── 2025-04-29_xxx.md  # 具体实验记录
├── 2025-05-01_yyy.md
└── ...
```

---

## 📝 记录原则

1. **可复现**：环境、依赖、代码必须完整记录
2. **可对比**：明确与论文/基线的对比指标
3. **可追踪**：每个实验必须链接到相关的 [[Research Question]] 或 [[Hypothesis]]
4. **诚实记录**：负面结果同样有价值，如实记录失败和意外发现

---

## 🏷️ 标签规范

所有实验文件必须包含：
- `#type/experiment`
- `#status/seedling`（进行中）或 `#status/budding`（有初步结果）或 `#status/evergreen`（已验证）

可选标签：
- `#area/memory`, `#area/planning`, `#area/multi-agent`, `#area/self-evolution`
- `#result/positive`, `#result/negative`, `#result/inconclusive`
