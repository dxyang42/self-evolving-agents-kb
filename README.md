## 📝 笔记原则

1. **原子化**：一个文件一个概念
2. **强制反向链接**：每个新笔记至少链接到2个已有节点
3. **我的理解必填**：每个概念笔记必须有"我的理解/批判"段落
4. **日期戳**：自进化领域变化极快，给笔记加 `created::` 和 `updated::`
5. **先深后广**：5篇精读+反复链接 > 50篇摘要
